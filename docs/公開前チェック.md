# 公開前チェック

新しいリプレイ小説やPCプロフィールを追加する前に、すべて確認します。

## シナリオ

- [ ] シナリオが完了している
- [ ] ユーザーが公開対象として指定している
- [ ] 未回収の秘密、GM専用情報、将来の展開を含んでいない
- [ ] 冒頭にネタバレ範囲が書かれている

## 参加者とPC

- [ ] 公開するPC名、台詞、プロフィールの範囲を確認した
- [ ] プレイヤー名は本人が許可した表示名だけになっている
- [ ] 本名、メールアドレス、アカウントID、チャットIDなどを除いた
- [ ] 内輪向けの情報や公開を望まない発言を除いた

## 原稿

- [ ] 生ログではなく、公開用のリプレイ小説になっている
- [ ] PCの選択や台詞の意味を変えていない
- [ ] 補足や創作上のつなぎを追加した場合、その扱いを確認した
- [ ] 誤字、表記、見出し、リンクを確認した

## 画像と付属資料

- [ ] 自作または公開許可を得た画像だけを使用している
- [ ] 画像内に個人情報や未公開情報が写っていない
- [ ] 必要なクレジットと利用条件を書いた

## GitHub公開

- [ ] 下書きや一時ファイルを含んでいない
- [ ] 非公開正本のファイルやAIキャッシュを含んでいない
- [ ] パスワード、トークン、ローカルパスを含んでいない
- [ ] 差分を確認し、公開対象だけになっている

GitHubでは、後からファイルを削除しても過去の履歴に残る場合があります。迷う情報は公開せず、確認が取れてから追加します。
