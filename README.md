# 星明かりの道しるべ 公開リプレイ

創作プロジェクト「妄想世界／星明かりの道しるべ」で遊んだ、完了済みシナリオのリプレイ小説を公開するためのリポジトリです。

参加したプレイヤーが物語を読み返せることと、公開を許可された範囲で作品世界を楽しんでもらうことを目的としています。

## 公開内容

- `replays/completed/`：公開済みのリプレイ小説
- `characters/`：公開を許可されたPCプロフィール
- `assets/`：公開権利を確認済みの画像や付属資料
- `docs/公開前チェック.md`：新しい作品を公開する前の確認事項

## 読む前に

- リプレイには、完了済みシナリオのネタバレが含まれます。
- GM専用情報、未完了シナリオ、未公開設定、生のチャットログは収録しません。
- プレイヤー名は、本人が公開を許可した表示名だけを使用します。

このリポジトリに明示的な許諾が記載されていない文章・設定・画像の著作権は、それぞれの権利者に帰属します。無断転載、再配布、学習データとしての再収集は許可していません。
