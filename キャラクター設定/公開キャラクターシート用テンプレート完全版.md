### 武器・防具・スキルの表示規則

```text
武器・防具
→ 名称のみ
→ 強化値0は何も付けない
→ 強化値1以上は名称直後へ「＋数値」
→ 攻撃力と防護点は名称へ付けない
→ 「強化」「強化値」という文字は冒頭表示へ出さない

スキル
→ 名前【威力表示】
→ 名前【数値】を使えるのはスキルだけ
```

例：

```text
武器：アステルブレード＋3
防具：防具一式＋3
スキル：星流剣術
```

```text
武器：紅焔の鉤爪
防具：出雲の軽鎧
スキル：狐火【10】　奥義・九尾紅蓮天【56】
```

詳細欄では、名称とは分離して記載します。

```text
アステルブレード
攻撃力：20
強化値：＋3

防具一式
防護点：5
強化値：＋3
```

強化値0の武器・防具は、詳細欄では`0`と記録して構いませんが、冒頭表示へ`＋0`を出しません。

## 公開情報の境界

このページは公開資料です。

公開を承認された次の情報だけを掲載します。

- 人物設定
- キャラクタービルダー登録情報
- 装備・所持品
- 現在状態
- 正史確定セッションの出来事
- 公開可能な人間関係
- 公開可能な累積統計

次の内容は混ぜません。

- 非公開正本だけに存在する秘密
- KP専用情報
- 物語進行中の未公開情報
- 比較シミュレーション
- 正史外
- IF資料
- 無効化・巻き戻された展開
- 本人が話していない過去の推測
- 画像だけから推測した設定
- 未確認の通貨・所持品・負傷
- 未成立の能力を成立済みとする記述

現在状態は、非公開正本と正史確定セッションの記録を優先します。
