# 登場人物

## カイト

中継都市近郊の村からやってきた、十五歳前後の新人冒険者。

黒髪と黒い瞳を持つ、少し痩せた少年。村を守れる人になりたいという思いから冒険者になった。素直で勢いがあり、教えられたことへ真剣に向き合う一方、認められたい気持ちが先走ることもある。

武器は直剣。今回が初めての実戦となる。

## シーリス・アルタイル

カイトの初回依頼へ同行する先輩冒険者。

白いコートと大剣を身につけ、カイトが自分で判断し、失敗から学べるよう必要以上には手を出さない。ただし、致命的な危険を見過ごすことはない。

# 本編

## 鳥の消えた森

浅い森の倒木のそばで、カイトは赤く縁取られた葉を指先で持ち上げていた。

「葉の裏に赤い筋……これかな？」

表面の色だけではない。葉の裏側を走る赤褐色の筋と、根元に近づくほど濃くなる茎の色。

シーリスから教わった赤縁草の特徴と一致している。

湿った土と青い草の匂いが混ざり、頭上では風に揺れる葉が静かな音を立てていた。

その音の中から、鳥の声だけが消えた。

ぱきり。

森の奥で枝が折れる。

シーリスが顔を上げた。

```text
【警戒｜シーリス】
80→7［イクストリーム成功］｜接近するゴブリン四体を発見
```

「カイト。草から手を離して、ゆっくり立って」

声は静かだった。

だが、彼女の右手はすでに剣の柄へ添えられている。

茂みの奥から、四つの小さな影が現れた。

錆びた短剣を握る小柄なゴブリン。

首に赤い布を巻き、欠けた剣を持つゴブリン。

片目の上に古い傷があり、鉈を引きずるゴブリン。

そして、節くれ立った棍棒を担ぐ、ひときわ大柄なゴブリン。

「ギャッ、ギャギャ……」

湿った笑い声が木々の隙間を這う。

シーリスが大剣を抜いた。

鋭い鞘走りの音に、四体の笑い声が一瞬だけ止まる。

「右の小さい一体だけ見て。追わない。私から離れないこと」

カイトは唾を飲み込み、腰の直剣を抜いた。

本物のモンスター。

村で聞いた話ではない。訓練用のカカシでもない。

「はい。僕が倒してみせます！」

恐怖を押し込むように叫び、落ち葉を蹴った。

## 最初の一撃

小柄なゴブリンが短剣を構える前に、カイトは肩口へ直剣を振り下ろした。

だが、ゴブリンは悲鳴に近い声を上げて地面へ身を投げた。

剣は頭上を通り過ぎ、細い枝を斬り落とす。

「えっ！」

```text
【攻防｜カイト→小柄な短剣持ちゴブリン】
攻50→35［レギュラー成功］｜先手＋／初実戦・悪い足場−
防30→12［ハード成功］｜身体を沈めて回避
結：防御側優勢｜斬撃をかわされ、カイトが前へ流れる
```

カイトの身体が前へ流れた。

そのすぐ向こうで、シーリスの大剣が風を裂く。

赤い布を巻いたゴブリンが木陰へ転がり込み、刃は幹を深く削った。乾いた破裂音とともに、木片が白いコートの周囲へ散る。

傷顔のゴブリンが側面から鉈を振るった。

「ギィッ！」

シーリスは身体をひねり、コートの裾だけを刃に持っていかせる。

続けて、大柄なゴブリンの棍棒が頭上から落ちた。

ごん、と重い音が森を震わせる。

シーリスは大剣を斜めに差し込み、棍棒の軌道を地面へ流した。湿った土が抉れ、泥と落ち葉が周囲へ跳ね上がる。

```text
【戦況進行】
シーリス→赤い布のゴブリン｜攻85→99［失敗］｜斬撃が木の幹を削る
傷顔の鉈持ちゴブリン→シーリス｜攻45→53［失敗］｜鉈がコートの裾をかすめる
大柄な棍棒持ちゴブリン→シーリス｜攻45→35［レギュラー成功］／防80→14［ハード成功］｜大剣で受け流す
```

その間に、小柄なゴブリンが起き上がっていた。

前へ流れたカイトの左肩へ、短剣が突き出される。

カイトは慌てて横へ跳んだ。

```text
【攻防｜小柄な短剣持ちゴブリン→カイト】
攻45→5［イクストリーム成功］｜攻撃直後の隙＋
防50→49［レギュラー成功］｜横へ回避／踏み込み過ぎ−
結：攻撃側優勢｜左肩へ浅く命中
```

布の裂ける音。

鋭い痛みが肩を走り、袖の裂け目から赤い線が滲んだ。

「痛っ！」

傷は浅い。

それでも、自分の血を見た瞬間、カイトの喉は強く締まった。

「カイト、そいつから目を離さないで！」

シーリスの声が森を切る。

カイトは直剣を握り直した。

## 逃げようとした赤い布

「今度こそ！」

カイトは小柄なゴブリンの武器腕を狙って剣を横へ振った。

ゴブリンは短剣を胸元へ引き、刃が届くぎりぎりの距離まで跳び下がる。

また空を切った。

小柄なゴブリンはすぐに反撃せず、大柄な仲間の位置を確かめるように首を動かした。

一方、木陰へ逃れた赤い布のゴブリンは、倒れた仲間がいないことを確かめながら、じりじりと森の奥へ足を向け始めていた。

「逃がさない」

シーリスが踏み込む。

大剣が低く走り、赤い布のゴブリンの胴を捉えた。

「ギッ……！」

欠けた剣が地面へ落ちる。

赤い布が落ち葉の上へ沈み、動かなくなった。

傷顔のゴブリンが甲高い叫び声とともに鉈を叩きつける。

シーリスは白いコートを翻し、刃の外へ抜けた。

鉈が空を裂き、勢い余ったゴブリンの足が湿った泥を滑る。

その隙に、大柄なゴブリンがカイトとシーリスの間へ割り込んだ。

棍棒が低く構えられ、二人を分断する。

```text
【戦況進行】
シーリス→赤い布のゴブリン｜攻85→72［レギュラー成功］／防35→43［失敗］｜逃走前に胴へ命中・戦闘不能
傷顔の鉈持ちゴブリン→シーリス｜攻45→44［レギュラー成功］／防80→7［イクストリーム成功］｜鉈を完全回避
大柄な棍棒持ちゴブリン｜カイトとシーリスの間へ移動
```

小柄なゴブリンは、強い仲間が近くに来たことで再び歯を剥いた。

「ギャアッ！」

短剣を左右へ振りながら、カイトの腹へ飛び込む。

カイトは逃げなかった。

両手で直剣を握り、迫る短剣へ刃を合わせる。

金属同士が甲高い音を立てた。

短剣が外へ弾かれ、切っ先がカイトの頬の横を通り過ぎる。

```text
【攻防｜小柄な短剣持ちゴブリン→カイト】
攻45→21［ハード成功］｜仲間の援護＋
防50→18［ハード成功］｜直剣で受け流す
結：防御側優勢｜短剣を外へ弾く
```

「防げた……！」

「そのまま相手を見て！」

## 初めて倒した相手

短剣を弾かれたことで、小柄なゴブリンの右腕が大きく外へ流れている。

カイトはその隙を見逃さなかった。

「そこだ！」

直剣を引き戻し、空いた胸元へ突き出す。

ゴブリンは後ろへ跳ぼうとした。

だが、遅い。

直剣の切っ先が胸元へ食い込んだ。

「ギャッ……！」

短剣が落ちる。

小柄な身体は数歩よろめき、倒木の根元へ崩れ落ちた。

```text
【攻防｜カイト→小柄な短剣持ちゴブリン】
攻50→44［レギュラー成功］｜防御直後の隙＋／左肩負傷−
防30→72［失敗］｜後退回避
結：カイト優勢｜胸元へ命中・戦闘不能
```

カイトはすぐに剣を引き抜いた。

手に残る感触が、訓練用のカカシとはまるで違う。

倒した。

自分の剣で、本物のゴブリンを。

だが、倒れた相手を見続けることはしなかった。

教えられたとおり、シーリスの方へ顔を上げる。

傷顔のゴブリンが、滑った足を踏み直しながら鉈を持ち上げていた。

シーリスの方が早い。

大剣が横へ走り、鉈ごと身体を薙ぎ払う。

傷顔のゴブリンは地面へ転がり、そのまま動かなくなった。

残る敵は、大柄な一体だけになった。

```text
【戦況進行】
シーリス→傷顔の鉈持ちゴブリン｜攻85→43［レギュラー成功］／防35→90［失敗］｜胴へ命中・戦闘不能
大柄な棍棒持ちゴブリン｜戦意70→32［ハード成功］｜仲間が倒れても逃走せず、カイトを狙う
```

## 逃げなかった最後の一体

仲間が三体倒れても、大柄なゴブリンは逃げなかった。

鼻息を荒く吐き、棍棒を両手で握り直す。

黄色く濁った目が、シーリスではなくカイトを捉えた。

自分より小さく、肩を負傷している少年。

仲間を倒した相手。

「グルル……ギャアアッ！」

大柄なゴブリンが走り出す。

棍棒が頭上へ振り上げられた。

シーリスが叫ぶ。

「カイト、来る！」

カイトは逃げなかった。

迫るゴブリンを、振り下ろされる棍棒を、最後まで目で追った。

棍棒が落ちる直前。

カイトは一歩だけ前へ踏み込み、ゴブリンの身体の横をすり抜けた。

空を切った棍棒が背後を通り過ぎる。

どん、と大木が揺れた。

```text
【攻防｜大柄な棍棒持ちゴブリン→カイト】
攻45→21［ハード成功］｜体格差・突進＋
防50→1［クリティカル］｜直前まで敵を見ていた＋
結：カイト優勢｜完全回避・大柄なゴブリンが木へ激突
```

大柄なゴブリンは勢いを止められず、肩から木の幹へ激突する。

枝から葉が降り注ぎ、驚いた鳥が一斉に飛び立った。

「今！」

シーリスの声が森を切り裂く。

ゴブリンの膝は曲がり、カイトへ背を向けたまま大きく体勢を崩している。

カイトは痛む左肩を庇いながら、その背後へ踏み込んだ。

「これで終わりだ！」

狙ったのは身体の中心ではない。

棍棒を支える脚。

直剣が膝裏へ走る。

ゴブリンは振り返ろうとしたが、足元の木の根に取られた。

刃が膝裏へ入る。

```text
【攻防｜カイト→大柄な棍棒持ちゴブリン】
攻60→56［レギュラー成功］｜背後・体勢崩れ＋／左肩負傷−
防35→100［ファンブル］｜振り返りながら防御
結：カイト優勢｜膝裏へ命中・転倒・棍棒落下
```

「ギャアアアッ！」

大柄な身体が傾き、棍棒が地面へ落ちた。

ゴブリンは片膝をつき、両手を土へつく。

その横を、シーリスが駆け抜けた。

白いコートが舞い、大剣が肩の高さまで持ち上がる。

倒れたゴブリンが、落とした棍棒へ手を伸ばした。

間に合わない。

重い刃が武器を持つ腕を打ち据え、ゴブリンの身体を地面へ押し倒した。

```text
【戦況進行】
シーリス→転倒した大柄なゴブリン｜攻85→12［イクストリーム成功］／防25→77［失敗］｜武器腕へ命中・戦闘不能
```

「ギッ……」

短い声を最後に、大柄なゴブリンは動かなくなった。

## 森へ戻った音

しばらく、誰も動かなかった。

森に残るのは、カイトの荒い呼吸と、葉の擦れる音。

遠くで、逃げていた鳥たちが警戒するように鳴いている。

シーリスは倒れた四体を順番に確認してから、大剣を下ろした。

「終わり」

その言葉を聞いて、カイトの肩からわずかに力が抜けた。

痛みはまだある。

左肩の傷は浅いものの、袖は裂け、血が細く流れていた。

シーリスは大剣の汚れを布で拭い、カイトのそばへ歩いてくる。

「腕は動く？」

カイトは顔をしかめながら、ゆっくり肘を曲げた。

「はい。痛いですけど……動きます」

シーリスは傷口を確認し、薬と包帯を取り出した。

「浅い。でも、今日は帰る」

「でも、赤縁草は見つけました」

「見つけたことと、安全に採れることは別」

薬が傷へ触れ、カイトの肩が小さく跳ねた。

包帯が巻かれていく。

森の奥で何かが枝を揺らし、カイトの身体が強張った。

シーリスも音の方を見る。

「四体だけとは限らない。ゴブリンがいたことを、ギルドへ報告する」

処置を終えると、彼女はカイトが落とし物をしていないか周囲を確かめた。

「最初は攻撃を外した。でも、次は相手を見て防いだ」

カイトは倒れた小柄なゴブリンへ視線を向ける。

「僕、倒せました」

「うん」

シーリスは短くうなずいた。

「最後まで目を閉じなかった。それがよかった」

カイトは直剣についた汚れを拭い、鞘へ収める。

赤縁草は、倒木のそばに残されていた。

依頼は終わっていない。

それでも二人は、静かになった森を中継都市へ引き返す。

冒険者の仕事は、敵を倒すことだけではない。

生きて帰り、何が起きたのかを報告するところまで続いている。

# 中間プレイリザルト

## 戦闘結果

- カイトは左肩に浅い切創を負った。
- カイトは短剣を持った小柄なゴブリンを戦闘不能にした。
- カイトは大柄なゴブリンの突進を完全に回避した。
- カイトは大柄なゴブリンの膝裏を斬り、転倒させた。
- シーリスは赤い布のゴブリン、傷顔のゴブリン、大柄なゴブリンを戦闘不能にした。
- ゴブリン四体は全員戦闘不能となった。
- 赤縁草は発見済みだが未採取。
- 周辺の安全を確認できないため、採取依頼は中断となった。

## カイトの状態

```text
左肩に浅い切創
応急処置済み
直剣を保持
歩行可能
```

## この戦闘で見えたカイトの人物像

- 初めての実戦でも、怖さを押して剣を抜いた。
- 攻撃を外しても、敵から目を離さなかった。
- 一度受けた攻撃への対応を、次の防御へ反映できた。
- 仲間が倒れても逃げない大柄な敵を前に、目を閉じず回避した。
- 自分だけで決着させようとせず、シーリスが動ける状況を作った。
- 負傷後も指示を聞き、戦闘終了後の帰還判断へ従った。
