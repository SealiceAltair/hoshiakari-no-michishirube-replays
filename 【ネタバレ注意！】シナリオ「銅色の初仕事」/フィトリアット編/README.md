# フィトリアット編『銅色の初仕事』

このフォルダーには、同じPC・同じ共通初期状態を使用して、複数の進行AIで実施した比較シミュレーションを保存しています。

## 正史として採用した版

- Claude Opus 4.8版
- セッション番号：`TRPG-0002`
- 扱い：正史確定
- この版で成立した出来事だけを、フィトリアット・マリアベルの継続履歴へ反映します。

## 比較用シミュレーション

- Claude Sonnet 5版：`TRPG-0003`／正史外
- GPT-5.6 Thinking版：`TRPG-0004`／正史外

比較用シミュレーションは削除せず、モデルごとの描写、判定処理、人物解釈、戦闘展開を比較する資料として保存します。

比較版でのみ発生した負傷、取得物、敵、NPC関係、技名、台詞、世界変化は、正史版の履歴へ混ぜません。

## 正史と番号の関係

セッション番号は、どの卓・シミュレーションかを識別するための番号です。番号が付いていること自体は、正史採用を意味しません。

```text
TRPG-0002
→ 正史確定

TRPG-0003
→ 比較シミュレーション／正史外

TRPG-0004
→ 比較シミュレーション／正史外
```
