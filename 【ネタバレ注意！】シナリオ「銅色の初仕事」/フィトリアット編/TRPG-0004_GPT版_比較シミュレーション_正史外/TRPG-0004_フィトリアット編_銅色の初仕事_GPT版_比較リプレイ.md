# 登場人物

## フィトリアット・マリアベル

十五歳の少女。

勢いと自信が先に立ち、堂々とした「ですわ」「のだわ」調で話す。自分の力を疑わず、特大剣「グラン」を単なる装備ではなく相棒として扱う。

気力を剣へ通す高度な武器強化を使える一方、冒険者としての採取、警戒、保存、応急手当は未経験に近い。

出自や家の事情については、自分から詳しく語ろうとしない。

## グラン

フィトリアットの主武器である特大剣。

本人は「相棒」と呼ぶ。長大で重量があり、狭い場所や近距離では取り回しに難しさを抱えるが、間合いと威力に優れる。

フィトリアットは気力を刀身へ通し、硬い対象や武器ごと斬り崩す戦い方をする。ただし、強引に気力を押し込めば疲労と隙が増える。

## 中継都市ギルドの受付員

二十代ほどに見える若い女性。

親しみやすく柔らかな物腰だが、危険職を扱う職員として曖昧な注意はしない。本人が話していない秘密や、確認できない出自を冒険者管理票へ勝手に書かない。

## 実技試験の担当者

ギルドの訓練場を管理する職員。

武器の威力だけでなく、攻撃手段を安全に制御できるか、周囲を巻き込まないかを確認する。

## シーリス

初回実地依頼へ同行する銀級冒険者。

白い髪の若い女剣士。普段は静かで少し近寄りがたく見えるが、冒険実務の話になると説明が具体的になる。

フィトリアットの代わりに正解を選ばず、致命的にならない範囲では本人へ判断と失敗を任せる。危険が現実になった時は、救援できる位置を保ちながら戦う。

# 本編

## 第一章　受付台へ来た大剣

中継都市の冒険者ギルドは、朝から人と紙と革の匂いで満ちていた。

濡れた外套を畳む者。依頼書を睨む者。腰の留め具を見せながら報告する者。奥では木箱が床を擦り、受付台の向こうでは羽根ペンが忙しなく走っている。

入口の扉が開いた時、そのざわめきが一度だけ細くなった。

少女が一人、長大な剣を背負っていたからだ。

剣先が床板へ触れないよう、フィトリアットは肩の角度を少し変えた。人の間を通る時には鞘を片手で押さえ、誰にもぶつけず、入口正面の受付台まで進む。

受付員が顔を上げる。

「ご依頼ですか？」

フィトリアットは胸を張った。

「冒険者登録に参りましたわ。フィトリアット・マリアベルですの」

続いて、背負っていた剣の柄へ軽く手を添える。

「こちらはグラン。わたくしの相棒ですわ」

受付員の視線が、少女と特大剣の間を一往復した。

「主な攻撃手段は、その剣で間違いありませんか？」

「当然ですわ。わたくしとグランを引き離して登録するおつもり？」

「いいえ。武器の種類を確認しただけです」

受付員は笑い、登録料が銀貨二枚であること、実技試験と最低限の講習が必要であることを説明した。

フィトリアットは用意していた銀貨二枚を台へ置く。

家名を聞いた受付員が、出身地や来訪経路について続けようとした時、フィトリアットの笑みがほんの少しだけ固くなった。

「細かな出の話は、依頼に必要な分だけで結構ですの。剣を扱えることと、仕事を果たせること。それで不足がありまして？」

受付員は問い詰めなかった。

「分かりました。出身と来訪経路は、本人が詳細を申告しなかったものとして扱います。必要になった時は、改めて確認しますね」

「話が早くて助かりますわ」

冒険者管理票には、確認できた事実だけが記された。

```text
名前：フィトリアット・マリアベル
年齢：15歳
主武器：特大剣「グラン」
戦闘手段：特大剣／気力による武器強化
出身：本人が詳細を申告せず
来訪経路：詳細未確認
登録料：銀貨2枚、支払い済み
```

## 第二章　銅色を得るための二撃

実技試験場には、大型獣系の丈夫な皮を張った訓練用カカシが据えられていた。

担当者はフィトリアットへ、試験の目的を簡潔に伝える。

「見たいのは、街の外で自分を守れる力と、その力を安全に扱えるかどうかです。周囲に人がいることを忘れないでください」

「威力と制御。両方をお見せすればよろしいのね」

フィトリアットはグランを抜いた。

長い刀身が空気を押し分ける。両手で柄を握り、足を開く。呼吸を一つ整えると、腕と肩を巡った気力が柄から刀身へ流れた。

光の刃が飛ぶことはない。

ただ、鋼の輪郭が一段だけ重く、硬くなる。

### 第一試技　気力を通した斬撃

```text
【判定結果】
成功値：70
出目：67
結果：レギュラー成功
```

成功値の理由：

```text
特大剣を主武器として扱う経験＋
気力を武器へ通す技術＋
威力を示す目的が明確＋
試験場で周囲が整理されている＋
大きな武器を狭い訓練区画で止める必要−
```

フィトリアットは肩から踏み込み、斜めにグランを振り下ろした。

刀身は丈夫な皮を裂き、その下の詰め物へ深く食い込む。カカシの支柱が大きく鳴ったが、根元から倒れることはなかった。

威力は十分。

ただし、刃を止めるためにフィトリアットの後ろ足が半歩滑った。

担当者は頷きながらも、まだ合格を告げなかった。

「力は分かりました。次は気力を抑え、印を付けた位置だけを斬ってください。深さより制御を見ます」

フィトリアットは一瞬だけ眉を上げた。

「グランに手加減をさせる試験ですの？」

「あなたが手加減できるかを見る試験です」

「よろしいですわ。相棒へ任せきりではないところも、お見せしますの」

### 第二試技　気力を抑えた制御斬り

```text
【判定結果】
成功値：75
出目：5
結果：イクストリーム成功
```

成功値の理由：

```text
一度目でカカシの硬さと距離を把握＋
目的が「深く斬る」から「指定位置で止める」へ明確化＋
フィトリアット自身が制御へ意識を切り替えた＋
気力使用を抑え、反動を減らした＋
```

二撃目は、一撃目より静かだった。

足音は一度。風を裂く音も短い。

グランの刃は白く引かれた印を正確になぞり、皮一枚と浅い詰め物だけを切って止まった。

切れ目の端は印からほとんど外れていない。

担当者がカカシを確認し、ようやく言った。

「合格です。威力は十分。制御はそれ以上に良い。ただし、気力を通す時は周囲へ合図を。味方はあなたの間合いを知りません」

フィトリアットはグランを持ち上げ、血振りの代わりに小さく刃を払ってから鞘へ戻した。

「心得ましたわ。グランの間合いを知らずに近づく方が悪い、と言いたいところですけれど」

一拍置き、少しだけ声を落とす。

「味方を斬っては仕事になりませんものね」

## 第三章　面談、講習、銅級留め具

受付台の脇にある小さな面談室で、フィトリアットは受付員と向かい合った。

質問は出自を暴くためではなく、依頼を任せられる人物かを知るためのものだった。

「冒険者になろうと思った理由を聞いてもいいですか？」

「わたくしとグランが、どこまで通じるのか確かめるためですわ」

「名声のため？」

「名が知られることを嫌う理由はありませんの。ですが、飾ってもらうためだけに剣を持っているわけでもありませんわ」

フィトリアットは椅子へ深く座らず、いつでも立てる姿勢のまま続けた。

「依頼があり、越えるべき場所があり、最後に結果が残る。冒険者の仕事は、試すにはちょうどよいと思いましたの」

「危険だと分かった時は、撤退できますか？」

「退くべき時には退きますわ。ただし、逃げるのではありませんの。次に勝つために下がるのだわ」

受付員は、その答えを否定しなかった。

「それで構いません。生きて報告することも仕事です。依頼書の危険度は答えではなく警告。包帯がない、帰り道を考えていない、相手が群れると知らない。そういう準備不足で人は死にます」

講習では、次のことが伝えられた。

- 依頼書の条件を読み落とさないこと。
- 現地で目的を達成しても、帰還し報告するまでは完了ではないこと。
- 危険度の低い相手でも、群れ、地形、毒、装備によって危険は変わること。
- 怪我を隠さず、応急手当を行うこと。
- 成果を傷めず、証拠を保全すること。
- 嘘の報告をしないこと。

最後に、受付員は小さな銅色の留め具を机へ置いた。

「フィトリアット・マリアベルさん。中継都市所属、銅級冒険者として登録します」

フィトリアットは留め具を取り上げ、光へ透かした。

金でも銀でもない。磨かれた銅の色。

それでも、ただの飾りではない。

自分の名前で受け取った、冒険者として最初の証明だった。

「悪くありませんわね」

口元へ浮かんだ笑みを隠すように、フィトリアットは留め具を剣帯へ付けた。

「いずれ、グランに見合う色まで上がってみせますわ」

フィトリアットが依頼書を二度見する。

「銅級の初仕事として、採取、保存、移動、報告まで確認する依頼です」

「グランの出番がありませんわ」

その時、隣から静かな声がした。

「葉っぱを斬るなら、たぶん出番はない」

振り向くと、白い髪の若い女剣士が立っていた。軽大剣を帯び、必要な荷物だけをまとめた鞄を背負っている。

「シーリス。今日の同行役」

「フィトリアット・マリアベルですわ。こちらはグラン」

「大きいね」

「水は？」

「ありますわ」

「包帯」

「ありますの」

「小さな刃物。採取袋。傷めないための布」

フィトリアットは一つずつ示す。小さな実用刃物を持っていることを見たシーリスは、そこで初めて小さく頷いた。

「グラン一本で全部やる人じゃなくて安心した」

「グランは万能ですけれど、成熟葉を十枚まとめて両断する依頼ではありませんもの」

「依頼書を読んでるなら大丈夫」

その短い返答が、少しだけ褒め言葉に聞こえた。

二人は中継都市の門を抜け、浅い森へ向かった。

## 第五章　葉を見つけるまでが採取

赤縁草は、一般的な傷薬の素材だ。

深緑の葉の縁に赤い筋があり、森林や林道脇、湿った木漏れ日の地面へ生える。若葉は薬効が弱く、傷薬には成熟葉を使う。葉を傷つけ、乾燥させ、黒ずませれば査定は落ちる。

説明を聞いたフィトリアットは、すぐ森の中を見回した。

「赤い縁ですのね。目立つなら簡単ですわ」

「まず自分で探してみて」

### 赤縁草の生息環境を探す

```text
【判定結果】
成功値：50
出目：63
結果：失敗
```

成功値の理由：

```text
赤い縁という外見情報を聞いている＋
観察する時間はある＋
薬草採取の実地経験が乏しい−
葉の色だけへ注意が偏っている−
```

フィトリアットは赤い色を探し、日当たりのよい場所ばかり見た。

赤い小花。枯れ葉の縁。虫食いで変色した草。

それらへ近づくたび、目的の草ではないと分かる。

「妙ですわね。赤いのに赤くありませんの」

「赤だけを見ると外す」

シーリスがしゃがみ、地面へ手をかざす。

「湿った土。木漏れ日。林道の脇。深緑の葉。その中に赤い筋がある」

フィトリアットは立ったまま探すのをやめた。

グランの鞘が枝へ当たらない位置へ身体をずらし、今度は地面の湿り、木の陰、草の密度を順に見る。

### 助言後の再探索

```text
【判定結果】
成功値：65
出目：38
結果：レギュラー成功
```

成功値の理由：

```text
生息環境の具体的な助言＋
一度目の失敗から探し方を変更＋
落ち着いて地面の条件を比較＋
採取経験そのものはまだ浅い−
```

林道から少し外れた木の根元。

湿った土へ重なるように、深緑の葉が広がっていた。その縁を、細い赤が走っている。

「ありましたわ」

フィトリアットは勝ち誇ったように振り返る。

シーリスは葉ではなく、その手元を見た。

「若いのと成熟したのが混じってる。どれを採る？」

「大きいものですわね」

「大きさだけじゃない。葉の厚みと赤い筋。黒ずみがないもの」

フィトリアットは成熟葉を選び、小さな刃物を当てた。

### 最初の株から成熟葉を採る

```text
【判定結果】
成功値：55
出目：59
結果：失敗
```

成功値の理由：

```text
対象となる成熟葉は判別できている＋
小さな刃物を用意している＋
初めての採取で力加減に慣れていない−
葉を押さえる位置が先端寄り−
```

切ろうとした瞬間、葉が逃げた。

押さえ直した指へ力が入り、赤い縁が折れる。もう一枚は切り口が潰れ、薬草として使えなくはないが、依頼条件の「傷めず」には届かなかった。

フィトリアットの眉が寄る。

「葉の方から逃げましたわ」

「根元に近い方を支える。引っ張らない。切る前に、袋を開いておく」

シーリスは答えを奪わず、手順だけを示した。

フィトリアットは傷んだ二枚を別へ分ける。

「失敗したものを混ぜないのだわね」

「うん。数をごまかすと、次から依頼を任されなくなる」

「そこまで落ちぶれてはいませんわ」

今度は膝をついた。

グランを手の届く位置へ横たえ、袋を開き、葉の根元を支え、刃を寝かせる。

### 方法修正後の採取

```text
【判定結果】
成功値：70
出目：13
結果：イクストリーム成功
```

成功値の理由：

```text
失敗した原因を理解＋
道具と袋を先に準備＋
葉を根元側で支える＋
速度より傷めないことを優先＋
```

六枚。

# チャッピー総評

フィトリアット・マリアベルの初仕事は、彼女の強さと未熟さが同じ一日の中ではっきり現れる冒険になった。

実技試験では、特大剣「グラン」の威力だけでなく、気力を抑えた精密な制御まで示した。一方、森へ出れば、薬草の生息環境を外し、葉を傷め、周囲への警戒を失い、武装したゴブリン二体の攻撃を受けている。剣士として強くても、冒険者としてはまだ新人であることが、ダイス結果によって公平に示された。

特に大きかったのは、被弾後の修正である。力を増やすのではなく、柄から刃へ気力を通す。長い剣を振り回すのではなく、敵の武器の短さと踏み込みを読む。彼女は失敗を消さず、その場で次の一手へ変えた。

最後に依頼を達成させたのは、二体を倒した腕力だけではない。予備の葉を採り、保存失敗を隠さず、未確認の巣を追わず、十枚と正確な報告を持ち帰った判断である。

この日、フィトリアットは英雄になったわけではない。

銅級冒険者として、最初の仕事を一つ終えた。

それは彼女が望む大きな物語の入口として、十分に重い一歩だった。
