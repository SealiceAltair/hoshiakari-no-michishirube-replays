# 本編

## 一、扉を押し開けて

中継都市の大通りから一本入った先、古びた木の扉と真鍮の看板を掲げたギルドがある。木と羊皮紙、わずかな鉄錆の匂いが漂う建物の中央へ、その少女は遠慮なく踏み込んだ。

背丈に迫るほどの特大剣を背負い、歩くたびに剣帯の金具が鈍く鳴る。

「――ここが冒険者ギルドですのね! 上等ですわ!」

受付カウンターの奥の女性職員が顔を上げる。丁寧な物腰と、危険職を扱う職員らしい芯の強さを併せ持つ人だった。

「いらっしゃいませ。本日はどのようなご用件でしょうか」

「冒険者登録ですわ! このフィトリアット・マリアベル、今日からここで名を上げますの!」

## 二、登録

「お名前を教えていただけますか」

「フィトリアット・マリアベル。それだけで十分ですわ」

出自や経歴には触れず、名だけを差し出す。受付員はそれ以上を尋ねず、台帳へ書き付けた。

「かしこまりました。登録料として銀貨2枚をお願いします」

フィトリアットは巾着から迷わず銀貨を二枚取り出し、カウンターへ置いた。

「これでよろしくて?」

「はい、確かに。では次に、実技試験へご案内します」

## 三、実技試験

裏手の試験場には、大角を持つ猪型モンスターの丈夫な毛皮で作られた訓練用カカシが据えられている。立ち会いの試験官が、腕を組んで待っていた。

「そいつが得物か。大きいな」

「ふふん、当然ですわ。これはグラン――わたくしの相棒。この子と組めば、大抵の壁は斬り拓けますの」

自信満々に振りかぶった一撃は、勢いだけが先行した。

```
【判定】成功値60（膂力と度胸はあるが、得物の重量をまだ制御しきれていない）
CC<=60 【実技試験：カカシへの一撃】
【判定結果】出目84・失敗
```

剣の重みに上体が持っていかれ、切っ先はカカシをかすめただけ。フィトリアット自身がたたらを踏んだ。

「――っ、な、なんですのこの子、今日はやけに重いですわ!?」

「得物のせいにするな。振りかぶりが大きすぎる。踏み込みと同時に重心を落とせ。剣を『投げる』んじゃなく『運ぶ』んだ」

素直に助言を取り入れ、姿勢を組み直す。今度は振りかぶりを浅くし、踏み込みと共に剣の重みごと前へ流し込むように構えた。

```
【判定】成功値65（助言を即座に反映。度胸と正しい重心移動が噛み合う）
CC<=65 【実技試験・再挑戦】
【判定結果】出目12・イクストリーム成功
```

唸りを上げた刃が、硬い毛皮ごとカカシの芯材まで一息に断ち割った。上半分がずるりと滑り落ち、乾いた木片の音だけが試験場に響く。

試験官はしばし黙り、それから小さく笑った。

「――才能あるな、お前。教われば化ける。合格だ」

「当然ですわ! グラン、よくやりましたわね!」

## 四、面談

「ご出身は?」
「――それはあまり、語りたくありませんわ。今のわたくしに必要な話ではありませんもの」

受付員は特に気にした様子もなく、話題を移した。剣を扱ってきた経験、依頼書の読み方、ギルド規則、死亡リスク、応急手当の重要性、危険度を過信しないこと、嘘の報告をしないこと――ひととおりの会話の中に、最低限の講習が自然に織り込まれていく。

冒険者になった理由を問われると、フィトリアットはにやりと笑った。

「決まっていますわ。この剣が振るうに値する舞台が欲しいのですの。それに――誰にも文句を言わせないくらい、名を上げてやりますわ」

## 五、銅級留め具

面談を終えた受付員が、剣帯へ銅色の留め具を取り付けた。

「これで、中継都市所属の銅級冒険者としてご登録完了です」

フィトリアットは腰元で鈍く光る留め具を見下ろした。

「銅、ですのね。……悪くありませんわ。すぐに次の色へ変えてみせますけれど」

## 六、初回依頼とシーリス

```text
依頼種別：採取
対象：赤縁草の成熟葉　必要数：10枚
場所：中継都市近郊の浅い森
条件：傷めず保存して当日中に納品
同行者：シーリス
```

ロビーで待っていたのは、軽装の剣士――銀級の留め具を下げたシーリスだった。フィトリアットの背の剣を見て、少し驚いたように瞬く。

「わ、大きい剣……。あなたが今日の新人さん? シーリスよ。よろしくね」

「フィトリアット・マリアベルですわ。よろしくしてさしあげますわ」

シーリスはきょとんとし、それからくすっと笑った。

「よろしくしてさしあげます、って……ふふ、面白い子ね。じゃ、行きましょうか」

## 七、浅い森

門から半刻ほど歩いた先、木漏れ日の落ちる浅い森。

「このあたりに赤縁草が生えてるはずよ。葉の縁が薄い赤みを帯びてるの、わかる?」

```
【判定】成功値45（説明を受けたばかりで判別経験は浅い）
CC<=45 【赤縁草の識別】
【判定結果】出目44・レギュラー成功
```

「――これ、ですわね? 縁がほんのり赤い」
「そう、それ! 筋がいいわね」

見分け方は掴んだが、いざ収穫する段になると――

```
【判定】成功値40（力任せに扱う癖があり、繊細な作業には不慣れ）
CC<=40 【採取：傷めず収穫する】
【判定結果】出目51・失敗
```

根元をつかんで引くと、勢い余って葉が裂けた。

「あっ……」
「根元から引っ張っちゃダメ。茎を挟んで、そっと折るの」

手本を見た直後、もう一度挑む。

```
【判定】成功値50（手本を見た直後だが、力の加減はまだ抜けきらない）
CC<=50 【採取・再挑戦】
【判定結果】出目64・失敗

【採取結果（中断時点）】
赤縁草　　　　×5
傷んだ赤縁草　×3
```

「……力、強すぎですわね、わたくし」
「うん、でも大丈夫。繰り返せば感覚は掴めるわ」

## 八、ゴブリンの群れ

```
【判定】成功値55（採取に集中し、周囲の警戒が疎かになっていた）
CC<=55 【森の異変を察知する】
【判定結果】出目62・失敗
```

草むらの奥で枝が折れる音。顔を上げた時にはもう、緑がかった影が三つ、四つと下草の間から現れていた。赤い布切れを腰に巻いた先頭の一体が、粗末な短剣を握って踏み出す。

```
【判定】成功値40（完全な奇襲。体勢を立て直す猶予に乏しい）
CC<=40 【奇襲直後：体勢を立て直す】
【判定結果】出目85・失敗
```

驚きによろけたところへ短剣が振り下ろされ、二の腕を浅く裂いた。

「フィトリアット!」

シーリスが叫びながら、奥の棍棒持ちへ斬り込む。フィトリアットは痛みに顔をしかめながらも、怯まず腰を落とした。

「――このくらい、ッ! かすり傷ですわ!」

**〈フィトリアット→赤い布切れの小柄なゴブリン〉**

```
攻58→18［ハード成功］狙：胴／踏み込み直後＋
防25→91［失敗］
結：フィトリアット優勢｜完全命中・戦闘不能
```

特大剣が唸りを上げて薙ぎ払われ、小柄なゴブリンは短剣ごと弾き飛ばされて動かなくなった。

「――当然ですわ!」

だが息をつく間もなく、大柄な棍棒持ちの一体が横合いから襲いかかる。

```
攻35→48［失敗］（攻撃側失敗のため防御判定なし）
```

棍棒は空を切った。踏み込みが浅く、間合いにすら届かなかった。

「――甘いですわ!」

腕から血が滴るのも構わず、フィトリアットは腹の底から気力を練り上げ、刀身へ流し込んだ。柄を握る手のひらに熱が集まり、刃全体がわずかに陽炎のように揺らぐ。

「グラン――全力全開ッ!!」

**〈フィトリアット「纏気一閃」→大柄な棍棒持ちゴブリン〉**

```
攻65→45［レギュラー成功］気力を纏わせた渾身の一閃＋／直後、腕と呼吸に反動－
防30→11［ハード成功］棍棒を盾に体を捻る
結：ゴブリン側がわずかに優勢｜致命打には至らず・棍棒破損・大きく怯む
```

会心の一撃――のはずだった。だが大柄なゴブリンは本能的に棍棒を盾のように立て、体を捻って急所を外した。棍棒は根元から弾け飛び、代わりに肩口へ深い一撃を受けたゴブリンが大きくよろめく。

「――嘘でしょう、防ぎましたの!?」

渾身の一撃を防がれた反動で、フィトリアットの腕は重く痺れ、息が荒くなる。その隙にシーリスが残る一体を的確に打ち倒した。

```
【判定】成功値45（仲間を二体失い、深手を負った個体が戦意を保てるか）
CC<=45 【残るゴブリンたちの士気】
【判定結果】出目87・失敗
```

肩を押さえて後ずさっていた大柄なゴブリンが、悲鳴じみた声を上げて身を翻す。残る一体も釣られるように、統制もなく森の奥へ駆け去っていった。

「――待ちなさ……ッ」
「フィトリアット、深追いは駄目! あなたも今、腕から血が出てるでしょう!」

シーリスが鋭く止めた。フィトリアットは荒い息のまま、グランを支えに何とか踏みとどまる。

「……くっ、逃げ足だけは達者ですわね」

## 九、撤退と応急処置

シーリスは手早く布を取り出し、フィトリアットの二の腕の傷を洗って縛った。

「浅いけど、ちゃんと処置しておかないと。……よく我慢したわね」

「これしき……平気ですわ。でも、あの大きいのに防がれたのは、正直ちょっと悔しいですのよ」

「あれは相手が上手く体を捻っただけ。あなたの一撃自体は、十分すごかったわ」

籠の中には、目標に届かない量の赤縁草。これ以上森に留まるのは危険が大きいと判断し、採取を切り上げた。

「……採取は、ここまでにしますわ。悔しいですけれど」

「正しい判断よ。行きましょう」

## 十、帰還・納品・報告

ギルドへ戻ったフィトリアットは、自ら受付へ籠を差し出し、事の次第を報告した。

「――赤縁草、規定の十枚には届きませんでしたわ。それと、安全区域のはずの森でゴブリンの群れに遭遇しましたの。二体はわたくしが、二体はシーリスが仕留めましたけれど、深手を負った一体を含む二体が森の奥へ逃走しましたわ」

受付員は籠の中身を確認し、傷の具合にも目を配りながら記録した。

「赤縁草は良品5枚、傷んだもの3枚――計8枚ですね。規定数には届きませんが、安全を優先して撤退されたのは適切なご判断です。ゴブリン群れの目撃情報、確かに承りました。こちらはギルドの警戒情報として記録します」

「――ふん。次はきっちり十枚、揃えてみせますわ」

「無理はなさらず。今日のところは、よく戻ってきてくださいました」

# 今回の特別評価

## NPCから見たPC

シーリスとフィトリアットの間で交わされた具体的な内心や評価文までは確認できないため、推測して固定しない。実際の交流から成立した人物関係だけを記録する。

フィトリアットは、渾身の一撃を防がれた際に「嘘でしょう」と本気で驚き、直後に「悔しい」と本人の口から漏らした。これは彼女が己の技量に強い自負を持ちながらも、結果を素直に受け止められる性質を持つことを示している。シーリスは深追いを止める際、はっきりとした口調で危険を告げており、新人の勢いを頭ごなしに否定せず、しかし譲らない態度で撤退判断を促した。

この関係性は、重大な恋愛感情や恒久的な師弟契約へ拡張しない。

# チャッピー総評

フィトリアット・マリアベルは、勢いと自信を先に立てて動くタイプの新人だった。実技試験では一度目の失敗をすぐに素直な軌道修正へつなげ、二度目でイクストリーム成功という鮮やかな結果を見せた。一方で、繊細な採取作業や周囲への警戒といった「地味な基礎」には苦戦し、二度の採取失敗、そして奇襲を許すという形でその粗さが露呈した。

戦闘では、小柄な相手には会心の一撃を通す実力を示しつつ、切り札である気力の一閃を大柄な相手にわずかに防がれるという結果になった。これは彼女の未熟さそのものというより、「ハードだが公平」な世界の中で、切り札にも絶対はないという教訓に近い。悔しさを素直に口にしながらも剣を手放さず、深追いの誘惑をシーリスの一声で抑えられたことは、彼女がまだ本人らしい危うさと同時に、指示に従える一面も持ち合わせていることを示している。

初仕事は条件付き達成に終わったが、彼女がこの先「グラン」と共にどこまで歩んでいくのか、続きが楽しみになる幕開けだった。
