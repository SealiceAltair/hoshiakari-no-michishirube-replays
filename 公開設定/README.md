# 公開設定

[公開資料トップへ戻る](../README.md)

「妄想世界／星明かりの道しるべ」から、公開を承認されたルールと設定をまとめています。

## 公開中

- [星みちTRPG 公開ルールブック](星みちTRPG_公開ルールブック.md)
  - プレイヤーが参照する共通判定、戦闘、HP、スキル、防御、迎撃、士気のルール
- [星みちTRPG 公開キャラクター・モンスターデータ](星みちTRPG_公開キャラクター・モンスターデータ.md)
  - 公開可能なキャラクターシート、派生戦闘値、通常ゴブリンの基準データ
- [星みちTRPG キャラクタービルドシミュレーター](https://sealicealtair.github.io/hoshimichi-character-sheet-builder/)
  - ポイント、装備、スキルを入力し、共有用Markdownを作る補助ツール

## 資料の境界

- 公開ルールブックは、非公開正本のPC向けルールに準拠した公開文面です。
- 公開データは、公開を承認された人物・モンスター情報だけを掲載します。
- ビルドシミュレーターは入力と試算の補助ツールであり、ルール正本ではありません。
- 入力内容やツールの出力は、非公開正本へ自動反映されません。
- 物語進行中の敵の内部値、未発見能力、秘密の目的、未公開の固有能力は掲載しません。
