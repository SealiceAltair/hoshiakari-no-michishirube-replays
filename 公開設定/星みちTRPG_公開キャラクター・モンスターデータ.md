# 星みちTRPG 公開キャラクター・モンスターデータ

[公開資料トップへ戻る](../README.md) / [PCキャラクター一覧](../PCキャラクター/README.md)

星みちTRPGで使用する公開ステータスと、主要人物・PC・通常モンスターの公開データです。

この資料はプレイヤー向けです。

敵の非公開数値、未公開設定、固有能力の秘密、判定式は掲載しません。

---

## ステータスの見方

### 基礎ステータス

健康な一般人を`基準`とします。

```text
基準
→ 一般人相当

■
→ 一般人より1ポイント高い
```

基礎ステータスの上限は15です。

表示順は次のとおりです。

1. 頑健
2. 持久
3. 筋力
4. 技量
5. 敏捷
6. 感覚
7. 知性
8. 精神

### 潜在能力

```text
―
→ ポイント配分なし

■
→ 潜在能力1ポイント
```

潜在能力の上限は10です。

潜在能力が高くても、対応する技、魔法、聖術を自動的に習得しているわけではありません。

表示順は次のとおりです。

1. 気力
2. 魔力
3. 聖力

### 実務技能

```text
―
→ 未習得

■
→ 心得あり

■■
→ 得意

■■■
→ 熟練
```

料理、整備、手当、植物、鉱物、解体の上限は3です。

`冒険`のみ上限5です。

冒険は作中の成功率を直接上げる技能ではありません。

次の二つへ作用する、星みちTRPG独自の対話演出技能です。

- チャッピーがPCの短い入力を描写へ展開する密度
- `/`を使用した時に返す公開情報と状況整理の詳しさ

冒険が高くても、PCが知らない秘密、敵の非公開数値、正解行動、自動成功は得られません。

### 体格

体格はクトゥルフ神話TRPGのビルドと同義の固定値です。

```text
一般的な人間
→ 体格1

通常ゴブリン
→ 体格0
```

体格は共通ポイントを消費しません。

### 別枠

次は共通ポイントへ含めません。

- 固有能力
- スキル
- 武器
- 装備

シーリスは、冒険そのものを心から好む少女です。

勇者や英雄になることよりも、一人の冒険者として依頼を受け、仲間と旅をし、生きて帰ることを望んでいます。

危険な場面では短く明確に警告しますが、同行者の判断や見せ場を奪う万能な案内役ではありません。

## 星流剣術

聖騎士から受け継いだ剣術です。

> 止める剣ではなく、流す剣。

剣の振り方だけでなく、次を一つの体系として扱います。

- 体捌き
- 足運び
- 重心移動
- 呼吸
- 気力の巡らせ方
- 攻撃の受け流し
- 流れる斬撃
- 突き
- 間合いの制御

攻撃を正面から力で受け止めるのではなく、軌道、力、身体の流れを利用して逸らします。

シーリスは、幼い頃の力任せな生存剣から、傷を受ける前に攻撃を流し、生きて帰るための剣へ戦い方を変えています。

通常の斬撃、突き、足運び、受け流しには技名を付けず、必要な場面で自然な剣戟として使用します。

## 武器

### アステルブレード［25］

シーリスが学園卒業時に聖騎士師匠から贈られた軽大剣です。

- 片手剣より長い
- 一般的な大剣より細身で軽い
- 白銀を基調とした刀身
- 黒革巻きの柄
- 青い星を思わせる装飾
- 斬撃、突き、受け流し、軌道変更に適する
- 左手を主導として扱う
- 鞘は右腰

## 装備

### 防具一式［15］

# ゴブリン

## 公開プロフィール

| 項目 | 内容 |
|---|---|
| 分類 | モンスター |
| 基本危険度 | ★ |
| 体格 | 0 |
| 行動 | 基本的に群れで行動する |
| 生息 | 洞穴などに巣を作る |
| 討伐証明 | 左耳 |

小柄な略奪型モンスターです。

単体なら一般成人男性でも勝てる程度ですが、通常は群れを作り、武器、防具、荷物、食料を奪って利用します。

ゴブリン依頼では、種族そのものよりも次の条件が危険度を左右します。

- 群れの数
- 巣の有無
- 武装
- 略奪経験
- 上位個体
- 待ち伏せ
- 周辺地形

この公開ステータスは、武器も防具も持たない通常個体の基準値です。

武装した個体、経験を積んだ個体、進化個体は別データとして扱います。
