# 公開リポジトリ編集ルール

このリポジトリは「妄想世界／星明かりの道しるべ」の公開用リプレイ置き場です。非公開の設定正本とは、履歴も内容も完全に分離して管理します。

## 最優先ルール

- 完了済みで、ユーザーが公開を承認したシナリオだけを扱う。
- 非公開正本から内容を自動転記しない。
- 未完了シナリオ、GM専用情報、将来の展開、未公開設定を入れない。
- 生のチャットログをそのまま入れず、公開用に整えたリプレイ小説だけを入れる。
- 本名、メールアドレス、アカウントID、チャットID、ローカルパスなどの個人情報を入れない。
- プレイヤーの表示名、PC設定、台詞、画像は、公開許可が確認できた範囲だけを載せる。
- 公開リポジトリ内で下書きを管理しない。削除してもGit履歴に残りうるため、完成稿だけを追加する。

## 編集方針

- 元の出来事、PCの選択、台詞の意味を勝手に変えない。
- 読みやすさのための誤字修正や章分けはできるが、創作上の補完は明示する。
- 候補、保留、妄想を確定事項として扱わない。
- シーリス・アルタイルや作品の核となるテーマを勝手に変更しない。
- 公開範囲が曖昧な情報は削らず、公開前の要確認として報告する。

## 新しいリプレイの追加手順

1. `docs/公開前チェック.md`を確認する。
2. `replays/000_リプレイ小説テンプレート.md`を複製して完成稿を作る。
3. `replays/completed/`へ配置する。
4. 必要なPCプロフィールだけを`characters/`へ追加する。
5. 個人情報、未公開情報、画像権利、ネタバレ表記を再確認する。
6. 変更ファイルと公開内容を報告し、ユーザーの確認後に公開する。

## Codexへの指示

- 毎回の報告冒頭に更新内容名を書く。
- 編集後は、変更ファイル、公開内容、除外した情報、要確認を報告する。
- コミットやプッシュは、ユーザーがその回の公開を明確に依頼した場合だけ行う。
- 非公開正本のファイル、内部索引、AIキャッシュ、作業台帳をこのリポジトリへ追加しない。
